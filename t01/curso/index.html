Quais são as matérias dadas no curso?

Aqui na aceleradora nós aprendemos inumeras coisas. Se você esta pensando que vai entrar em um curso Básico de Programação, você está redondamente enganado. Aqui aprendemos desde as primeiras noçoes básicas de programação, até a gerência de projetos.

  • Algoritmos (instrução passo-a-passo para realizar qualquer tarefa.)
  • Portugol(Uma linguagem de programação de fácil manuseio em Português para iniciantes.)
  • HTML 5 (Para construção de websites.)
  • CSS (Para dar estilo em página HTML)
  • JavaScript (Linguagem de programação client-side.)

Como são as aulas?

As aulas na Aceleradora são fora do padrão, se você pensa que o professor vai ficar a aula inteira enchendo a lousa de matéria, está redondamento enganado de novo. Aqui as coisas são diferentes, aulas são feitas com dinâmicas de programação, tais como:

  • Aulas em pareamentos;
  • Aulas criativas;
  • Aulas de Português;
  • Aulas de Inglês;
  • Brincadeiras;
  • Jogos introdutivos;
